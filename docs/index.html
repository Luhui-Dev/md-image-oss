03 / 04

为什么是 md-image-oss

五条克制的、彼此独立的设计选择。

  • F.01

    多格式支持

    .md · .mdx · .html，一套工具搞定。

  • F.02

    安全解析

    跳过代码块、<script>、<style>、HTML 注释 —— 不动文字。

  • F.03

    四种图片语法

    Markdown、HTML <img>、引用式、Obsidian wikilink，任意混用。

  • F.04

    高质量压缩

    JPEG 渐进 q85、PNG 无损、WebP method=6；GIF / SVG 原样保留。

  • F.05

    内容哈希去重

    SHA-256 命名 —— 同一张图永不重复上传，跨文件共享。

  • F.0X

    更多 —— 视用户反馈而定。

04 / 04

快速开始

三行命令，从写完到发布。

terminal · ~/blog
# 1. 安装
$ pip install md-image-oss
# 2. 处理文件，输出 article.oss.md
$ md-oss article.md --env-file .env
# 3. 原地覆盖
$ md-oss article.md -i --env-file .env

需要先在 .env 里配置阿里云 OSS 凭证 —— 完整说明见 仓库 README ↗
